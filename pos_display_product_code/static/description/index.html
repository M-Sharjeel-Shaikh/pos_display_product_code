POS – Display Internal Reference (Default Code)

Quickly identify products in the POS by showing their internal references alongside product names.

🔍
Internal Reference Visibility

Displays product default codes beside product names in the POS interface.

🚀
Fast Product Identification

Enables quicker selection of similar or barcode-like products during sales.

⚙️
No Configuration Needed

Just install the module and it works automatically within your existing POS workflow.

How It Works

Simple steps to activate internal reference display in POS.

Step 1: Install the Module

Locate and install the “POS – Display Internal Reference in Product List” module from the Odoo Apps.

Step 2: Enable Option

Please check the option in the Point of Sale settings.

Step 3: Use During Sales

Cashiers can now easily identify products using internal references during transactions.
